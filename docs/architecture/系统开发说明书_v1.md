## 四、API 接口清单

### 4.1 对话接口
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/v1/chat/stream | SSE流式对话，返回多类型事件流 |

### 4.2 画像接口
| 方法 | 路径 | 说明 |
|------|------|------|
| GET  | /api/v1/profile/{user_id} | 获取用户八维画像 |
| POST | /api/v1/profile/{user_id} | 更新用户画像 |

### 4.3 学习路径接口
| 方法 | 路径 | 说明 |
|------|------|------|
| GET  | /api/v1/path/{user_id} | 获取当前学习路径 |
| POST | /api/v1/path/plan | 规划新路径 |
| POST | /api/v1/path/{user_id}/progress | 更新节点进度 |

### 4.4 资源接口
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/v1/resources/generate | 生成指定类型学习资源 |

### 4.5 日志接口
| 方法 | 路径 | 说明 |
|------|------|------|
| GET  | /api/v1/logs/session/{session_id} | 获取会话Agent操作日志 |

## 五、部署说明

### 5.1 后端启动
```bash
cd d:\software_APP\Socrates-Cube
.\.venv\Scripts\activate
# （可选）设置讯飞星火凭证
$env:SPARK_APP_ID="your_app_id"
$env:SPARK_API_KEY="your_api_key"
$env:SPARK_API_SECRET="your_api_secret"
# 初始化数据库
python scripts/init_db.py
# 导入课程文档
python scripts/ingest_docs.py
# 启动服务
uvicorn src.loopse.main:app --reload --host 0.0.0.0 --port 8000
```

### 5.2 前端启动
```bash
cd frontend
npm install
npm run dev
# 访问 http://localhost:5173
```

## 六、Agent 协作流程

```
用户输入
    │
    ▼
OrchestratorAgent（意图识别）
    ├─── RetrieverAgent（三库联合检索：course_docs / protocol_specs / misconceptions）
    ├─── ProfilerAgent（八维画像更新）
    ├─── DiagnosisAgent（三层认知诊断）
    ├─── ResourceGeneratorAgent（按需生成 doc/exercise/code）
    └─── PathPlannerAgent（知识图谱路径规划）
    │
    ▼（各 Agent 结果封装为 SSE 事件）
前端 useChatSSE() 消费：
    ├─── chatStore（消息+流token+诊断结果）
    ├─── userStore（画像更新→雷达图）
    ├─── resourceStore（资源卡片）
    └─── pathStore（路径时间轴）
```
