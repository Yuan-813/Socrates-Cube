# 环境初始化说明

> 适用系统：Windows 10/11、macOS 12+、Ubuntu 22.04+  
> Python 版本要求：**Python 3.10**（严格，不兼容 3.11+ 的某些依赖）

---

## 快速启动（3步）

```bash
# 1. 克隆仓库
git clone <仓库地址> && cd Socrates-Cube

# 2. 创建虚拟环境并安装依赖
python3.10 -m venv .venv
# Linux/macOS:
source .venv/bin/activate
# Windows:
.venv\Scripts\activate

pip install -r requirements.txt

# 3. 配置环境变量并验证
cp .env.example .env
# 编辑 .env 填入真实 API Key 后执行：
python scripts/verify_env.py
```

成功输出：`环境检查通过 ✅`

---

## 详细步骤说明

### Step 1：确认 Python 版本

```bash
python3.10 --version
# 期望输出：Python 3.10.x
```

若未安装 Python 3.10，请前往 https://www.python.org/downloads/ 下载安装。  
Windows 用户推荐使用 `py -3.10` 代替 `python3.10`。

### Step 2：创建虚拟环境

```bash
# Linux/macOS
python3.10 -m venv .venv
source .venv/bin/activate

# Windows (PowerShell)
py -3.10 -m venv .venv
.venv\Scripts\Activate.ps1
```

激活成功后命令行前缀会出现 `(.venv)`。

### Step 3：安装依赖

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

> 若 `chromadb` 安装失败，尝试先执行：`pip install setuptools wheel`

### Step 4：配置环境变量

```bash
cp .env.example .env
```

用文本编辑器打开 `.env`，填入以下必填项：

| 变量名 | 获取方式 |
|--------|---------|
| `XUNFEI_APP_ID` | 注册讯飞开放平台 → 控制台 → 创建应用 |
| `XUNFEI_API_KEY` | 同上 |
| `XUNFEI_API_SECRET` | 同上 |

其余变量（DATABASE_URL、CHROMA_DB_PATH）保持默认值即可在本地开发使用。

### Step 5：初始化数据库

```bash
python scripts/init_db.py
# 期望输出：数据库初始化完成 ✅
```

### Step 6：启动后端服务

```bash
uvicorn src.loopse.main:app --reload --host 0.0.0.0 --port 8000
```

访问 http://localhost:8000/health 应返回：

```json
{"status": "ok", "version": "0.1.0"}
```

## 常见问题

| 问题 | 解决方案 |
|------|---------|
| `uvicorn: command not found` | 确认虚拟环境已激活，或执行 `pip install uvicorn` |
| `chromadb` 安装报错 | 执行 `pip install setuptools wheel` 后重试 |
| `.env` 中 API Key 无效 | 检查讯飞控制台应用状态是否为"审核通过" |
| Windows 下 `Activate.ps1` 权限报错 | 以管理员身份执行 `Set-ExecutionPolicy RemoteSigned` |
